Основные команды

Команда Описание
/reg [пароль] Регистрация аккаунта на сервере
/l [пароль] Авторизация на сервере
/changepass [старый] [новый] Сменить пароль от аккаунта
/ab toggle Отключить сообщения в чате от ботов
/spawn Телепортация на спавн
/suicide Самоубийство (если застряли где-то)
/help Открыть меню помощи
/kit старт Получить набор предметов для начала игры
/menu Открыть главное меню сервера
/balance Проверить свой баланс

Команды телепортации

Команда Описание
/tpa [никнейм] Отправить запрос на телепортацию к игроку
/tpahere [никнейм] Отправить запрос на телепортацию игрока к себе
/tpaccept [никнейм] Принять запрос на телепортацию
/tpdeny [никнейм] Отклонить запрос на телепортацию
/spawn Телепортироваться на спавн

Команды экономики

Команда Описание
/balance Проверить свой баланс
/pay [никнейм] [сумма] Перевести монеты другому игроку
/shop Открыть магазин сервера
/ah Открыть аукцион сервера
/ah sell [цена] Выставить предмет на аукцион
/ah cancel Снять с продажи свой предмет с аукциона
/ah collect Забрать просроченные предметы с аукциона
/ah amount Проверить сколько предметов вы выставили на аукцион
/jobs Открыть меню работ для заработка валюты

Команды ОдинБлока

Команда Описание
/ob Телепортация на свой остров
/ob count Посмотреть свою фазу и количество сломанных блоков
/ob phases Посмотреть все фазы на сервере
/ob reset Сбросить свой остров и начать заново
/ob sethome [имя] Установить точку дома на острове
/ob homes Посмотреть список точек дома
/ob deletehome [имя] Удалить точку дома
/ob setname [имя] Установить название острова
/ob settings Открыть настройки острова
/ob respawnBlock Восстановить блок, если он исчез
/ob ban [никнейм] Забанить игрока на своем острове
/ob unban [никнейм] Разбанить игрока на своем острове
/ob banlist Посмотреть список забаненных игроков
/ob expel [никнейм] Выгнать игрока с вашего острова
/ob visit Открыть меню посещения других островов
/ob visit configure Настройки посещения вашего острова
/ob visit setlocation Установить точку появления для посетителей
/ob likes Посмотреть/поставить лайк острову
/ob near Посмотреть близлежащие острова
/ob team Управление командой острова
/ob team invite [никнейм] Пригласить игрока в команду острова
/ob team help Посмотреть все команды для управления командой

Команды коммуникации

Команда Описание
/t [никнейм] [сообщение] Отправить личное сообщение игроку
/tell [никнейм] [сообщение] Отправить личное сообщение игроку
/r [сообщение] Ответить на последнее личное сообщение
/me [действие] Отправить сообщение о своем действии
/telegram Получить ссылку на Telegram канал сервера
/discord Получить ссылку на Discord сервер

Команды развлечений

Команда Описание
/sit Сесть на блок или на землю
/lay Лечь на блок или на землю
/crawl Ползти
/sex [никнейм] Отправить предложение для создания ребенка с кастомными торгами
/head Меню декоративных голов
/itf Меню настройки рамок (невидимость или цвет)
/music Меню с музыкой (требуется ресурспак сервера)
/skin [никнейм] Установить себе скин другого игрока
/skin url [ссылка] Загрузить скин по ссылке (.png)
