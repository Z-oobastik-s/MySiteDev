Плагины сервера

На нашем сервере установлено множество плагинов, каждый из которых добавляет уникальные возможности и улучшения игрового процесса. Ниже представлен список основных плагинов по категориям.

Защита территории Экономика и торговля Улучшение геймплея Дополнительные возможности

Защита территории

GriefPrevention

Позволяет игрокам создавать защищенные приваты для своих построек, предотвращая кражи и разрушения.

  • Создание приватов с помощью золотой лопаты
  • Настройка доступа для других игроков
  • Защита от огня и взрывов
  • Расширение территории с игровым прогрессом

LWC

Плагин для защиты отдельных блоков, таких как сундуки, печи, двери и другие интерактивные блоки.

  • Приватные сундуки и другие контейнеры
  • Пароли для доступа к заблокированным блокам
  • Публичный доступ с ограничениями
  • Защита механизмов от посторонних

WorldGuard

Позволяет администраторам создавать защищенные регионы и устанавливать правила для различных территорий.

  • Глобальные и локальные флаги для регионов
  • Безопасные зоны спавна
  • Защита от различных типов урона
  • PvP и другие настройки игровых механик

Экономика и торговля

Vault

Базовый плагин для экономики, который обеспечивает работу других плагинов с валютой сервера.

  • Интеграция с другими экономическими плагинами
  • Единая валютная система
  • Поддержка транзакций между игроками
  • API для работы с экономикой

ShopGUI+

Удобный магазин с графическим интерфейсом, где игроки могут покупать и продавать различные предметы.

  • Интуитивный интерфейс магазина
  • Динамические цены на предметы
  • Категории товаров для удобства
  • Поиск предметов в магазине

QuickShop

Позволяет игрокам создавать собственные магазины для продажи предметов другим игрокам.

  • Создание магазинов с помощью табличек
  • Пополнение товаров в магазинах
  • Настройка цен для своих товаров
  • Защита магазинов от краж

Jobs Reborn

Система профессий, позволяющая игрокам зарабатывать деньги, выполняя различные действия в игре.

  • Разнообразные профессии (шахтер, лесоруб, фермер и др.)
  • Повышение уровня навыков
  • Бонусы за достижение определенных уровней
  • Возможность работать в нескольких профессиях

Улучшение геймплея

EssentialsX

Набор основных команд и функций, необходимых для комфортной игры на сервере.

  • Телепортация, дома и варпы
  • Личные сообщения и каналы чата
  • Команды для управления инвентарем
  • Настройка погоды и времени

Backpack

Добавляет возможность иметь дополнительный инвентарь для хранения предметов.

  • Дополнительное место для хранения
  • Разные размеры рюкзаков
  • Возможность улучшения рюкзака
  • Сохранение содержимого при смерти

mcMMO

Добавляет RPG-элементы в игру, позволяя игрокам прокачивать различные навыки.

  • Прокачка навыков (рубка, копание, бой и т.д.)
  • Специальные способности для каждого навыка
  • Бонусы к добыче ресурсов
  • Рейтинг игроков по навыкам

OneBlock

Режим игры, где вы начинаете с одного блока и постепенно расширяете свой остров.

  • Фазы развития острова
  • Случайная генерация ресурсов
  • Уникальные дропы и мобы
  • Достижения и прогресс

Дополнительные возможности

VotingPlugin

Позволяет игрокам голосовать за сервер на различных площадках и получать за это награды.

  • Интеграция с популярными сайтами голосования
  • Автоматическая выдача наград
  • Статистика голосования
  • Ежемесячные топы голосующих

DiscordSRV

Интеграция сервера с Discord, синхронизация чата и уведомления о важных событиях.

  • Связь игрового чата с Discord
  • Уведомления о входе/выходе игроков
  • Статус сервера в Discord
  • Команды сервера через Discord

Dynmap

Интерактивная карта мира сервера, доступная через веб-браузер.

  • Карта всех миров сервера
  • Отображение игроков на карте
  • Маркеры и варпы на карте
  • Чат интегрированный с картой

Plan

Подробная статистика сервера и отдельных игроков, доступная через веб-интерфейс.

  • Статистика игрового времени
  • Графики активности сервера
  • Детальная информация о каждом игроке
  • Анализ производительности сервера
