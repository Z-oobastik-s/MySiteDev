Команды Minecraft 1.21.4

Основные команды

Команда Описание Пример
/gamemode Изменение режима игры /gamemode creative
/tp Телепортация /tp @p 0 100 0
/give Выдача предметов /give @p diamond 64
/effect give Наложение эффектов /effect give @p speed 60 2
/time Управление временем /time set day

Команды для администраторов

Команда Описание Пример
/op Назначить оператором /op PlayerName
/ban Забанить игрока /ban PlayerName Причина
/kick Выгнать игрока /kick PlayerName Причина
/whitelist Управление белым списком /whitelist add PlayerName
/gamerule Настройка правил игры /gamerule doDaylightCycle false
/weather Управление погодой /weather clear
/kill Уничтожение сущности /kill @e[type=zombie]
/summon Призыв сущностей /summon creeper ~ ~ ~
/xp Управление опытом /xp add @p 100 levels

Новые команды в 1.21.4

Команда Описание Пример
/trialkey Управление пробными ключами для Minecraft Realms /trialkey generate
/wind Управление ветром в игре /wind speed set 10
/item Модификация предметов /item replace entity @p weapon.mainhand with diamond_sword
/transfer Перемещение игрока на другой сервер /transfer PlayerName server1
