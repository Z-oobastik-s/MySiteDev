Пример использования ServerStatus

Демонстрация работы компонента статуса сервера

Как использовать компонент ServerStatus

Для использования компонента ServerStatus на вашей странице, следуйте этим шагам:

1. Подключите CSS-файл в заголовке страницы:

<link rel="stylesheet" href="css/components/server-status.css">

2. Добавьте элемент-контейнер в HTML:

<div id="serverStatus"></div>

3. Подключите JavaScript-файл компонента перед закрывающим тегом body:

<script src="js/components/ServerStatus.js"></script>

4. Инициализируйте компонент:

<script>
    document.addEventListener('DOMContentLoaded', function() {
        // Создание экземпляра компонента
        const serverStatus = new ServerStatus({
            container: '#serverStatus',
            serverAddress: 'play.zoobastiks.ru',
            serverPort: 25565,
            apiEndpoint: 'https://api.mcsrvstat.us/2/',
            refreshInterval: 60000,
            autoRefresh: true
        });
        
        // Инициализация компонента
        serverStatus.init();
    });
</script>

Параметры конфигурации

Компонент ServerStatus принимает следующие параметры:

Параметр Тип По умолчанию Описание
container string '#serverStatus' CSS-селектор контейнера для отображения компонента
serverAddress string 'play.zoobastiks.ru' Адрес Minecraft-сервера
serverPort number 25565 Порт Minecraft-сервера
apiEndpoint string 'https://api.mcsrvstat.us/2/' URL API для получения статуса сервера
refreshInterval number 60000 Интервал обновления статуса в миллисекундах (1 минута)
autoRefresh boolean true Автоматическое обновление статуса
