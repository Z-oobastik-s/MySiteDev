Документация по серверным командам

Здесь собраны команды, недоступные обычным игрокам:

Команда Описание Пример
/data Управление NBT данными /data get entity @p
/execute Выполнение сложных команд /execute as @e[type=zombie] at @s run tp @s ~ ~10 ~
/function Запуск функций /function namespace:path/to/function
/scoreboard Управление системой очков /scoreboard objectives add Money dummy
/summon Призыв сущностей /summon lightning_bolt ~ ~ ~
